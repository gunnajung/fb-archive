# GitHub Pages 배포 방법

## 준비물
- GitHub 계정 (github.com)
- Mac Terminal

## 단계별 방법

### 1단계: GitHub에서 새 Repository 만들기
1. https://github.com/new 접속
2. Repository name: `fb-archive` (원하는 이름)
3. Public 또는 Private 선택
4. **README 추가 없이** "Create repository" 클릭

### 2단계: Terminal에서 이 폴더로 이동
```bash
cd ~/Downloads/fb-archive-online
```

### 3단계: 배포 스크립트 실행
```bash
bash DEPLOY_TO_GITHUB.sh
```

### 4단계: GitHub Pages 활성화
1. https://github.com/[사용자명]/fb-archive/settings/pages
2. Source → "Deploy from a branch"
3. Branch → main, / (root)
4. Save 클릭

### 5단계: 완료!
약 3-5분 후 아래 URL에서 접근 가능:
```
https://[사용자명].github.io/fb-archive/
```

---
**주의**: Repository를 Private으로 만들면 GitHub Pages는 유료 플랜(Pro) 필요.
Public Repository로 만들면 무료로 사용 가능.
